# Alsa (Advanced Linux Sound Architecture)

## Configuration

Checking USB device

```bash
udevadm monitor --udev
```

Write you device path ex:

```
/devices/pci0000:00/0000:00:1d.0/usb2/2-1/2-1.2/2-1.2:1.1`
```

Find device info

```bash
udevadm info --path=/your/device/path --attribute-walk
```
