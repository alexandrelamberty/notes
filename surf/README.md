# Surf

## Surf

Ctrl+G Go to URL
Ctrl+y Yank the URL

## Tab

Ctrl-Shift-Enter Open a new tab
Ctrl-Shift-j/l Go previous / next tab
Ctrl-[0..9] Go to the nth tab
Ctrl-Tab Toggle between two most recent tab
Ctrl-q Close tab.
